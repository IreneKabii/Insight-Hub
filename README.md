# 📊 Insight-Hub: Executive Business Intelligence Dashboard

## 📌 Strategic Overview
**Insight-Hub** is a high-fidelity Business Intelligence (BI) platform built using **R Shiny**. Unlike static reports, this tool bridges the gap between raw data and executive strategy by providing real-time, interactive visualizations of complex business metrics.

This project showcases my ability to manage the full data lifecycle: from **SQL-based data refinery** to **statistical modeling in R** and final **UI/UX deployment**.

## 🚀 Key Business Features
* **Interactive KPI Scorecards:** Dynamic "Value Boxes" for instant tracking of Revenue, User Growth, and Operational Efficiency.
* **Predictive Scenario Modeling:** Built-in regression modules that allow stakeholders to simulate business outcomes using reactive sliders.
* **Deep-Dive Filtering:** Multi-variable filtering (Region, Department, Date) powered by a high-performance R backend.
* **Automated Data Pipeline:** Replaces manual spreadsheet workflows with a reproducible, script-based ETL process.

## 🛠 Technical Stack
- **Engine:** `R` (version 4.0+)
- **Dashboard Framework:** `shinydashboard`, `bslib`
- **Visualization:** `plotly` (interactive), `ggplot2` (static precision)
- **Data Wrangling:** `tidyverse`, `lubridate`, `scales`
- **Interactivity:** `DT` (DataTables for searchable reporting)
